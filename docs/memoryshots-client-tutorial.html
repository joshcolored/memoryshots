MemoryShots

Client Tutorial and Event Admin Guide

A QR event photo sharing system for weddings, birthdays, christenings, corporate events, anniversaries, and private celebrations.

Scan Snap Share Approve Download

1. Login To Your Admin Account

  1. Open the MemoryShots website.
  2. Click Login.
  3. Enter your admin email and password.
  4. Click Login.

After logging in, you will be redirected to the Dashboard.

2. Create An Event

  1. From the Dashboard, click Create event.
  2. Fill in the event title, event type, photo limit, cover image, event date, and settings.
  3. Click Create event.
The system automatically creates a random event link slug. You do not need to type the event URL manually.

3. Download The QR Poster

  1. Open the event from the Dashboard.
  2. Find the QR code panel.
  3. Click Poster PNG.
  4. Print the poster and place it around the venue.

Suggested locations: entrance, guest tables, registration table, photo booth area, stage, or screen area.

4. Share The Event Link

Use the Copy button to copy the event link, or click Guest link to open it.

You can share the link through Messenger, Viber, WhatsApp, email, or an event group chat.

5. Guest Photo Upload Flow

  1. Guest scans the QR code.
  2. Guest enters their name or nickname.
  3. Guest taps Take photo or Upload.
  4. Guest chooses a filter, if desired.
  5. Guest previews the photo.
  6. Guest taps Upload.

Each guest is limited to 16 photos per event.

6. Approve Photos

Uploaded photos are hidden until approved by the admin.

  1. Open the event from the Dashboard.
  2. Scroll to the Photos section.
  3. Click Approve to make a photo public.
  4. Click Hide to remove a photo from the public gallery.
  5. Click Delete to permanently remove a photo.

7. Public Gallery And TV Mode

The public gallery is available at /event/event-slug/gallery.

Only approved photos appear publicly.

TV mode is available at /event/event-slug/gallery?tv=1.

TV mode automatically slides photos every 3 seconds and includes a queue list with thumbnails, guest names, and capture dates.

8. Download All Photos

  1. Open the event from the Dashboard.
  2. Click ZIP.
  3. The system downloads all event photos as a ZIP file.

9. Delete An Event

Only delete an event when you are sure.

Deleting an event removes the event details, guest records, photo records, guestbook messages, and stored photos.

10. Best Practices

Before Event

  • Create the event early.
  • Test the QR code using your phone.
  • Print multiple QR posters.
  • Test guest upload and admin approval.
  • Test TV mode if using a screen.

During Event

  • Keep the admin dashboard open.
  • Approve good photos regularly.
  • Use TV mode to encourage guest uploads.
  • Download the ZIP after the event.

11. Troubleshooting

Camera does not open: Use the HTTPS website link. Camera access may not work on non-secure HTTP links.

Guest cannot upload: The event may be inactive, the guest may have reached 16 photos, the file may be too large, or the file type may be unsupported.

Photo does not show publicly: The photo must be approved by the admin first.

TV mode does not update: Refresh the TV mode page and check that the backend service is running.

12. Quick Admin Checklist

  • Login works
  • Event created
  • QR poster downloaded
  • QR poster tested
  • Guest upload tested
  • Admin approval tested
  • Gallery tested
  • TV mode tested
  • ZIP download tested
